Le fichier dossier courant contient le code source du TP4-P1 du cours MGL869c écrit par Théo WEIMANN.
C'est un projet maven.
Il est possible de lancer les test unitaires avec le script de 'test'.
Il est également possible de générer un executable .jar en lançant le script 'package'.
Tel qu'il est dans le repository, le programme execute une fonction type qui offre une couverture du code casi complète et affiche le résulat des opérations dans la console.
Pour lancer le package on peut utiliser la commande 'java -jar <nom du fichier jar>'.
